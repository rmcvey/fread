fread

Javascript CSV and Image File Reader

View the Project on GitHub rmcvey/fread

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

fread is a javascript microlibrary that will show previews of selected CSV and image files. Additionally, you can hook into the library and allow users to manipulate data prior to uploading.

installation

bower install fread

usage

var fread = new fread({
  in: '#file-input',
  out: '#table-output',
  before: function(evt){
    // perform some action before render
  },
  after: function(rows){
    // perform some action with rows array
  }
});
